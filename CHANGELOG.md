# Changelog

## 0.1.0

- LAN-first static file serving with terminal QR codes
- Native file watching and browser live reload
- SPA fallback and extensionless HTML URLs
- Safe capability-based filesystem access and hidden-file protection
- Styled directory listings with file sizes
- CORS, custom headers, cache-control, and request logging
- Custom 404 pages and optional browser launch
- Strict or automatic port selection
